## Quick Download Instructions

### Using Unsplash:
1. Go to https://unsplash.com/
2. Search for the image using the search terms above
3. Click on the image you like
4. Click "Download free" button
5. Rename the file to match the filename above
6. Place in the `images` folder

### Using Pexels:
1. Go to https://pexels.com/
2. Search for the image
3. Click "Free Download"
4. Select "Original" size
5. Rename and save to `images` folder

## Image Optimization (Optional but Recommended)

After downloading, you can optimize images for web:
- Use online tools like TinyPNG (https://tinypng.com/) or Squoosh (https://squoosh.app/)
- Target file size: Under 200KB per image
- Format: JPG for photos
